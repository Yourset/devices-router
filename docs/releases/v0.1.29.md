# Devices Router v0.1.29

## 键盘专用稳定版

- 暂时停用跨电脑鼠标控制、鼠标拦截和基于鼠标活动的自动切换。
- 旧配置即使曾开启鼠标功能，升级后也会被统一关闭。
- 两台电脑版本短暂不一致时，主电脑会忽略旧版副电脑发送的鼠标活动消息。
- 跨电脑键盘继续可用：`Ctrl+Alt+2` 切到副电脑，`Ctrl+Alt+1` 切回主电脑。
- `Ctrl+Alt+Esc` 本地紧急释放继续保留，断网时仍然有效。
- 界面明确标记鼠标功能暂时不可用，移除鼠标开关、灵敏度按钮和后端启用接口。

## 验证

- Python：33 个测试通过。
- Rust Clippy、TypeScript 与 Vite 构建通过。
