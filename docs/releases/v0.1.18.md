# Devices Router v0.1.18

## Changes

- Fixed arrow keys and related navigation keys in apps that require Windows extended-key input, including uTools-style launch/search panels.
- Applies the extended-key flag to PageUp, PageDown, End, Home, arrow keys, Insert, Delete, Windows keys, NumLock, right Ctrl, right Alt, and numpad divide.

## Notes

This keeps normal text and IME input on the virtual-key path while making navigation keys more compatible with apps that inspect extended-key metadata.
