# Devices Router v0.1.13

本版本重点把“能用”继续推进到“更适合普通用户测试”。

## 主要变化

- 新增设置页，集中管理启动偏好、安全发现、游戏模式和主题。
- 新增启动默认模式：
  - 沿用上次
  - 主电脑
  - 副电脑
  - 不自动启动
- 运行模式、键盘目标、主题、鼠标灵敏度按钮增加明确选中态。
- 鼠标跟随新增三档预设：稳定、平衡、灵敏。
- 游戏模式会关闭自动鼠标切换，降低游戏时误抢输入的风险。
- 网络诊断增加本机 LAN IP、键盘端口、更新端口检测。
- 日志区支持重复日志折叠，减少刷屏。

## 验证

npm.cmd run build
cargo test
