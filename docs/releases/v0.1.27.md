# Devices Router v0.1.27

## 修复

- 修复副电脑光标贴住左边缘后坐标不再变化，导致鼠标无法划回主电脑的问题。
- 返回判断改为同时使用“继续向左的输入意图”和副电脑真实左边界，不再依赖光标坐标必须继续减少。
- 主电脑连接断开、远端退出或鼠标发送失败时，立即按 fail-open 策略恢复本机输入。

## 安全能力

- 新增本机紧急键 `Ctrl+Alt+Esc`，在低级键盘 hook 中优先处理，不会转发到副电脑。
- 紧急释放会解除本机键盘和鼠标拦截，并补发远端鼠标按键和修饰键释放。
- 托盘新增 `立即释放控制（Ctrl+Alt+Esc）`。
- 首页和鼠标页新增 `立即回到主电脑` 按钮和明确的安全提示。

## 验证重点

- 主电脑右边缘划入副电脑。
- 副电脑左边缘继续向左划回。
- 控制副电脑时按 `Ctrl+Alt+Esc` 强制返回。
- 控制副电脑时断开网络或退出副电脑程序，主电脑自动恢复输入。
