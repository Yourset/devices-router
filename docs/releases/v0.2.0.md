# Devices Router v0.2.0

## 主要变化

- 一台主电脑可同时连接两台副电脑。
- 使用持久设备 ID 区分副电脑，同设备重连会替换旧连接，第三台不同设备会被拒绝。
- 鼠标仍由 Logi Flow 负责；Devices Router 仅根据三台电脑最后一次鼠标活动切换键盘。
- 30ms 防抖保留最后活动者，不丢失高频切换请求。
- 从副电脑 A 切到 B 前先向 A 发送修饰键和鼠标按钮松开；当前目标断线或发送失败会立即回主电脑。
- 总览页显示本机和两个副电脑槽位，支持设备别名和指定设备手动切换。
- 托盘提示显示当前键盘目标设备名称。

## 兼容与升级

1. 先在主电脑安装 v0.2.0。
2. 再在两台副电脑安装同一安装包并启动副电脑模式。
3. 新旧混用时，只允许一台没有设备 ID 的 legacy 副电脑。
4. 端口仍为 TCP 8765、UDP 8766、TCP 8767。

## 安全边界

- 跨电脑鼠标转发保持禁用。
- `Ctrl+Alt+Esc` 始终由主电脑本地处理。
- 自动化测试不会启动真实键鼠钩子。
