# Devices Router v0.1.26

## 新功能

- 不依赖 Logitech Flow，普通 Windows 鼠标也能从主电脑右边缘划入副电脑。
- 转发鼠标移动、左右中键、拖拽、垂直滚轮和横向滚轮。
- 控制副电脑时拦截主电脑的鼠标按键和滚轮，避免两台电脑同时被点击。
- 从副电脑左边缘向左划即可返回主电脑。

## 安全边界

- `v0.1.26` 默认开启跨电脑鼠标控制，仍可在“鼠标跟随”页手动关闭。
- 游戏模式会强制停止鼠标注入和本地拦截。
- 返回主电脑、断开连接或停止服务时，会补发鼠标按键释放事件。

## 验证

- `cargo test`：51 项 Rust 测试。
- `npm.cmd run build`：TypeScript 和 Vite 构建。
- `npm.cmd run build:lan-update`：Windows 安装包和局域网更新包。
