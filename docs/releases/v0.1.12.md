# Devices Router v0.1.12

## Highlights

- Faster mouse-follow switching.
- Faster UI status refresh.
- Bidirectional heartbeat for more accurate connection state.
- Clear/copy/export logs in the desktop UI.
- LAN update package generation for remote PCs.
- Bilingual Chinese/English documentation.

## Performance Changes

- UI status refresh: `1000ms -> 200ms`
- Host mouse polling: `30ms -> 20ms`
- Remote mouse report interval: `80ms -> 40ms`
- Switch debounce: `150ms -> 80ms`
- Host priority cooldown: `120ms -> 60ms`
- Heartbeat interval: `1000ms -> 300ms`

## Notes

This release targets Windows-to-Windows Logitech Flow companion usage. Logitech Flow still handles mouse movement; Devices Router forwards keyboard input and infers the active machine from mouse activity.
