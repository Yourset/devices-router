# Devices Router v0.1.19

## Changes

- Fixed host-side stuck modifier keys when switching control to the secondary PC, which could make normal keys trigger shortcuts after returning to the host.
- Sends local modifier key-up events before suppressing host keyboard input for remote control.
- Uses scan-code based extended-key injection for navigation keys, improving compatibility with uTools-style panels that do not reliably accept virtual-key arrow events.

## Notes

If a remote app still ignores injected keys, confirm whether that app is running elevated and restart Devices Router as administrator on the same PC.
