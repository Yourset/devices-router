# Devices Router v0.1.33

## 主界面稳定性

- 修复周期状态刷新每秒重建整页，导致主界面滚动后自动弹回顶部的问题。
- 状态刷新改为更新已有节点；标签内容仅在实际状态变化时替换，编辑主电脑 IP 时保留输入焦点。
- 主界面、侧栏和普通功能面板全部禁止滚动，日志文本框是当前唯一滚动区域。
- 窗口最小尺寸调整为 `880 × 680`，窄窗口继续保持双栏，避免生成超出窗口的纵向长页面。
- 网络地址与诊断信息改为双栏展示，并压缩面板间距，保证最小窗口下主要控制项可见。

## 验证

- 新增桌面布局契约测试，锁定局部状态刷新、滚动白名单、最小窗口高度和版本一致性。
- Rust、Python、Clippy、TypeScript 与 Vite 构建均需通过后发布。
