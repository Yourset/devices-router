# Devices Router v0.2.3

## 中文

- 主电脑会在每张副电脑卡片上显示实际控制连接的实时延迟。
- 副电脑会显示到主电脑的实时延迟。
- 延迟通过现有 TCP 控制连接每 500ms 测量一次，不使用 ICMP Ping，也不建立额外连接。
- 显示值采用 75% 旧值与 25% 新样本的平滑结果；连接中尚无样本时显示“测量中”。
- 断线或重连会清除旧样本，不会把旧连接的延迟显示到新会话。

## English

- The host shows live control-channel latency for each connected remote computer.
- A remote computer shows its live latency to the host.
- RTT is measured every 500ms over the existing TCP control connection, without ICMP or extra sockets.
- The displayed value uses a 75/25 smoothed sample; new connections show “measuring” until the first reply.
- Disconnecting or reconnecting clears stale latency samples.
