# Devices Router v0.1.32

## 键盘自动跟随加速

- 将主电脑会话轮询从 `10ms` 提升到 `5ms`。
- “平衡”档调整为：主电脑检测 `10ms`、副电脑上报 `15ms`、主电脑优先冷却 `25ms`、切换防抖 `30ms`。
- 将未显示的本机释放保护从 `1000ms` 降到 `120ms`，解决鼠标已经移动到副电脑但键盘仍要等待的问题。
- `Ctrl+Alt+Esc` 和“立即释放”使用独立的 `1000ms` 安全窗口，不受自动跟随加速影响。
- “灵敏”档调整为 `5ms / 10ms / 15ms / 20ms`，“稳定”档调整为 `20ms / 60ms / 100ms / 120ms`。
- 跨电脑鼠标转发和鼠标拦截继续保持关闭。

## 验证

- 自动跟随延迟预算回归测试通过。
- Rust、Python、Clippy、TypeScript 和 Vite 构建通过。
