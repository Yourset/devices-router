# Devices Router

Keyboard following companion for Logitech Flow.

Logitech Flow moves your mouse between computers. Devices Router lets a normal keyboard follow that movement.

## Quick Summary

- Windows -> Windows
- Ready-to-use `.exe` installer
- No Node.js, Rust, Python, or command line needed for normal users
- Host PC captures keyboard input
- Remote PC receives simulated keyboard input
- Mouse activity decides where the keyboard should go
- LAN auto-update for the remote PC
